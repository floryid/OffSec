Module 3: Cyber Kill Chain

3.1 Understanding the Cyber Kill Chain

The Cyber Kill Chain is a model that describes the stages of a cyber attack:

  • Reconnaissance: Gathering information about the target through various methods like OSINT, network scanning, and social engineering.
  • Weaponization: Preparing exploits and malicious payloads tailored to target vulnerabilities.
  • Delivery: Transmitting the weaponized payload to the target through methods like phishing emails or compromised websites.
  • Exploitation: Executing the exploit code to take advantage of vulnerabilities.
  • Installation: Installing malware or backdoors for persistent access.
  • Command & Control: Establishing communication channels for remote system control.
  • Actions on Objectives: Achieving the attack goals like data theft or system damage.

3.2 Practical Examples

Real-world examples and case studies of the Cyber Kill Chain in action:

Example: Web Application Attack

  • Reconnaissance: Using tools like Shodan to identify vulnerable web servers
  • Weaponization: Creating SQL injection payloads
  • Delivery: Submitting malicious input through web forms
  • Exploitation: Executing SQL injection to access database
  • Installation: Uploading web shells for persistence
  • C2: Using the web shell to control the server
  • Actions: Extracting sensitive data
