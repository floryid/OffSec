Modul 3: Cyber Kill Chain

3.1 Memahami Cyber Kill Chain

Cyber Kill Chain adalah model yang menjelaskan tahapan serangan siber:

  • Reconnaissance: Mengumpulkan informasi tentang target melalui berbagai metode seperti OSINT, pemindaian jaringan, dan rekayasa sosial.
  • Weaponization: Mempersiapkan exploit dan payload berbahaya yang disesuaikan untuk menarget kerentanan.
  • Delivery: Mengirimkan payload yang telah dipersiapkan ke target melalui metode seperti email phishing atau situs web yang telah disusupi.
  • Exploitation: Mengeksekusi kode exploit untuk memanfaatkan kerentanan.
  • Installation: Menginstal malware atau backdoor untuk akses berkelanjutan.
  • Command & Control: Membangun saluran komunikasi untuk kontrol sistem jarak jauh.
  • Actions on Objectives: Mencapai tujuan serangan seperti pencurian data atau kerusakan sistem.
Mulai Belajar

3.2 Contoh Praktis

Contoh nyata dan studi kasus Cyber Kill Chain dalam aksi:

Contoh: Serangan Aplikasi Web

  • Reconnaissance: Menggunakan tools seperti Shodan untuk mengidentifikasi server web yang rentan
  • Weaponization: Membuat payload SQL injection
  • Delivery: Mengirimkan input berbahaya melalui form web
  • Exploitation: Mengeksekusi SQL injection untuk mengakses database
  • Installation: Mengunggah web shell untuk akses berkelanjutan
  • C2: Menggunakan web shell untuk mengontrol server
  • Actions: Mengekstrak data sensitif
